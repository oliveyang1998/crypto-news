# 如何注册币安交易所

币安交易所是全球最大的数字货币交易平台，其交易量位居全球第一。同时，币安交易所也支持中国大陆用户的注册和身份认证。接下来，我将为你演示如何注册币安交易所以及如何在币安交易所购买比特币。

# **一、中国大陆注册币安流程**

你需要准备以下材料：

1. 中国大陆的邮箱或手机号
2. 非美国、加拿大、新加坡的IP地址，推荐使用日本、韩国的IP

**点击下方的链接注册一个币安账户：**

[https://accounts.binance.com/zh-CN/register](https://accounts.binance.com/zh-CN/register?ref=36457687)

选择自己的邮箱或手机号码进行注册。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/035fad1940_1697180723-11111111111111.jpg)

填写邮箱或手机号，输入币安邀请码36457687，即可享受百分之二十的返现比例（20%的手续费优惠）。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/880bb046d0_1705561912-111.jpg)

填写验证码，点击提交。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/9238aa10a9_1705561891-222.jpg)

按照以下要求设置密码。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/7711f0ac98_1697181616-44444444.jpg)

# **二、下载币安APP**

注册好账号后，系统会提示你进行身份认证。身份认证需要在手机上进行，因此你需要下载币安APP：

安卓版下载：**<https://vlink.cc/okxchina>**

苹果手机下载需要海外APPLE ID，你可以查看下面的视频进行注册。

接下来需要设置账户安全，绑定邮箱和手机号。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/6ca5a97e52_1697182987-5555555555.jpg)

以下3个安全项都很重要，建议全部进行设置。身份验证APP是安全等级最高的，不易被黑客盗取，类似于银行的U盾。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/97ae59d3ae_1697183357-666.jpg)

# **三、购买USDT流程**

接下来，点击左上角的【一键买币】按钮。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/2c9af628af_1697183590-7777.jpg)

输入你想要购买的金额，即可看到你能买到的USDT数量。点击【0手续费购买】按钮。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/934dcc5ca6_1697183835-888888888.jpg)

选择自己习惯的交易方式，点击“确认购买”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/ed8308e181_1697184056-9999999999.jpg)

接下来付款部分容易出错，需格外注意：

1. 付款时必须是本人付款，使用他人身份付款可能导致账号被冻结。
2. 付款时不要添加任何备注，否则会付款失败。不要添加“买币”、“USDT”、“买币款”等字样。
3. 注意核实收款人的姓名是否与订单上的一致。
4. 付完款后再点击“我已付款请放币”按钮，未付款时无需点击。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/7982b92803_1697184434-aaa.png)

# **四、购买虚拟货币流程**

点击右上角【钱包】按钮，找到下拉框内的“资金账户”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/159d182e3c_1697184921-BBBB.png)

1. 点击右上角【划转】按钮。
2. 注意划转方向：将“资金账户”里的币划转到“现货账户”。
3. 注意币种选择USDT。
4. 数量点击“全部”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/e412b71da7_1697185774-ccc.jpg)

现在可以进行交易。点击左上角【交易】按钮，选择现货交易。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/b6c54a8f07_1697186104-dd.jpg)

在交易页面，点击右侧红框内的“USDT”按钮，才能进行USDT交易。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/68fd16d59c_1697186642-eeee.jpg)

以购买狗狗币DOGE为例：

1. 在搜索框内输入“DOGE”。
2. 点击“DOGE/USDT”，这样就能用我们手里的USDT购买DOGE。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/06e18289e6_1697187005-ff.jpg)

1. 选择“市价”按钮（市价即按照市场上最新的价格购买）。
2. 向右拖动成交额下方的滑杆，选择你要购入的DOGE数量。
3. 点击“买入DOGE”按钮。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/2e2130e55b_1697187331-gggg.jpg)
