# **C2C交易流程怎么操作？**

交易步骤：
1. 仔细阅读商家的交易提醒（如实名付款、不要备注等）
2. 点击「0手续费购买」生成订单
3. 点击「获取付款详情」查看商家收款信息
4. 按要求转账给商家
5. 转账完成后点击「我已支付」
6. 等待商家确认收款并放币

**务必按商家要求操作**，如实名付款、不添加备注。我见过新手因操作不当导致交易延误。

# **买到的USDT怎么用来交易？**

买到的USDT默认在「资金账户」中，如需交易比特币等币种，需先划转到「交易账户」。

操作步骤：
1. 点击右下角「资产」
2. 选择「划转」
3. 从资金账户划转USDT到交易账户
4. 点击「交易」进入现货交易
5. 选择交易对，如BTC/USDT
6. 选择「市价委托」或「限价委托」
7. 输入购买金额，点击「买入」

市价委托立即按当前价格成交，限价委托设定理想价格等待成交。我常用市价委托快速交易，限价委托用于抄底。

# **链上转账怎么操作？**

如果需将其他交易所或钱包的币转到欧易，使用链上转账功能。

充币操作：
1. 首页点击「链上充币」
2. 选择币种（如USDT）
3. 选择网络（推荐TRC20，手续费仅1个USDT）
4. 复制充币地址
5. 在其他平台发起提币，粘贴此地址
6. 确保网络类型一致（如均为TRC20）

**网络类型必须一致**，TRC20对TRC20，ERC20对ERC20，否则币会丢失。

提币同理，在「资产」页面选择「提币」，输入目标地址即可。
